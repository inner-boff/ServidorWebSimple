Sobre el código 404

HTTP1​2​ 404 Not Found o HTTP 404 No Encontrado es un código de estado HTTP que indica que el host ha sido capaz de comunicarse con el servidor, pero no existe el recurso que ha sido pedido. Este error no debe ser confundido con «servidor web no encontrado» o errores similares en los que se indica que no se ha podido realizar la conexión con el servidor.
